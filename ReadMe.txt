
Introduction:
	FastKnock is an efficient algorithm to identify all possible knockout strategies for growth-coupled overproduction of biochemical(s) of interest.

This repository used to include code to implement the Fastknock algorithm published in Microbial Cell Factories 2024 (PMID: 38287320). 
Outdated: In 2021, the repository was moved to https://github.com/leilahsn/FastKnock.
